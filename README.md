# Octave, Unison, Fifth and Fourth - harmonic Eb

`Tonic` and `Dominant` can make two intervals.
When `Tonic` is below `Dominant` - this is `Fifth`.
When `Tonic` is above `Dominant` - this is `Fourth`.
